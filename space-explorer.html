CONTRIBUTION SUMMARY


Description

Solo-developed educational space exploration game designed for kids. Players become astronauts navigating stylized planetary systems, collecting samples, and researching them in a futuristic lab to uncover fun, educational facts.

Project Ownership

  • Solo-developed an educational space exploration game using Unity and C#.
  • Designed, programmed, and produced all game systems, architecture, and content independently.

Architecture & Systems

  • Implemented event-driven architecture using ScriptableObject-based Event Channels for decoupled system communication.
  • Built a modular Service Locator system via ScriptableObjects to manage runtime services (e.g., save, UI, audio).
  • Developed a highly generic and extensible Save System with automatic registration of saveable objects.

Content Creation with AI

  • Created all visual assets including planets, UI, and characters using AI tools (e.g., Leonardo AI, MidJourney).
  • Generated game music and sound effects with AI-driven audio synthesis tools.
  • Produced animations using AI-assisted pipelines for sprite-based movement and effects.
