PROJECT OVERVIEW


Description

A full-stack daily productivity app for structuring your day, managing personal goals, and building better habits — with AI-powered daily coaching insights. Penguin Daily enables users to set quests, plan their day in time blocks, track habit streaks, and receive personalized coaching tips generated by OpenAI based on their daily plan and reflection.

Key Features

  • AI Daily Coaching: OpenAI-generated personalized coaching tip each day, based on your scheduled time blocks, completed tasks, mood, and end-of-day reflection.
  • Day Planning: Schedule time blocks throughout the day, attach quests to each block, and toggle completion as you go.
  • Quest Management: Create and manage two types of quests — Simple Goals for one-off tasks and Daily Tracks for recurring habits.
  • Progress Tracking: Log events (complete, fail, partial) to update quest points and maintain detailed progress history.
  • Daily Habit Streaks: Track daily entries with streak counting and per-day logging for consistent habit building.
  • Mood & Reflection: Log your mood and a short reflection at the end of each day to feed into AI insights.
  • Quest Categories: Organize quests using built-in or custom categories for better management.
  • Dashboard Analytics: View aggregated statistics, recent activity, and personal progress overview.
  • JWT Authentication: Secure user authentication and authorization with token-based system.

Technical Architecture

  • Backend (NestJS): RESTful API with modular architecture separating concerns across authentication, quests, categories, events, and dashboard modules.
  • Database (PostgreSQL): Relational database with TypeORM for entity management, migrations, and complex queries.
  • Frontend (Next.js 13+): Server-side rendering with App Router, React Server Components, and Server Actions for seamless API communication.
  • Type Safety: Full TypeScript implementation across both frontend and backend for enhanced developer experience and code reliability.
