Corely — Modern Gym Management SaaS Platform

Full-stack ASP.NET Core MVC platform for gyms, studios, and coaches to manage memberships, branches, classes, and billing. Corely is a web-based gym management system designed for multi-branch fitness studios that centralizes everything — member registration, class scheduling, payments, and analytics — into one easy-to-use system.

Key Features

  • Dashboard: Real-time analytics and KPIs displaying member count, revenue, attendance, and class utilization with monthly trend comparisons.
  • Members Management: Register and manage members with personal details, membership plans, attendance tracking, and complete member history.
  • Staff Management: Hire and manage trainers, staff, and managers with role-based permissions and performance tracking.
  • Classes Management: Create, edit, and manage fitness classes with instructor assignments and capacity limits.
  • Schedule: Comprehensive scheduling system for classes, trainer availability, and member bookings with conflict detection.
  • Billing: Stripe integration for secure payments, automated invoice generation, and payment tracking with cash/manual payment support.
  • Membership Plans: Create custom membership plans by duration, branch access, time restrictions (e.g., morning-only), and pricing tiers.
  • Multi-Branch Support: Manage multiple gym locations from one dashboard with isolated analytics and custom branch-level settings.
  • Role-Based Access: Owner, Manager, Staff, Trainer, Member roles with fine-grained permissions and data isolation per gym.

Technical Architecture

  • Multi-Tenant System: Data isolation by TenantId (gym) with EF Core global filters to enforce tenant scope
  • Modular Design: Clean separation between business modules (Membership, Billing, Analytics)
  • Database: Entity Framework Core with SQL Server for scalable data management
  • Security: Role-based Identity system with comprehensive access control

Tech Stack

  • Frontend: ASP.NET Core MVC, Razor Views, custom CSS with modern minimal design
  • Backend: ASP.NET Core 9.0, Entity Framework Core
  • Database: SQL Server with multi-tenant architecture

Design & User Experience

  • Modern, minimal dashboard design created with Figma AI
  • Dark mode support for enhanced user experience
  • Fully responsive layout with intuitive sidebar navigation
  • Optimized for both desktop and mobile management tasks

Business Impact

  • Streamlines gym operations by consolidating all management tasks into one platform
  • Reduces administrative overhead with automated billing and invoice generation
  • Provides actionable insights through real-time analytics and reporting
  • Scales efficiently to support multiple gym locations and franchises
  • Enhances member experience through streamlined registration and payment processes
